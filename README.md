# Projeto - Engenharia de Dados e Big Data

Este repositório foi criado para armazenar atividades e projetos desenvolvidos na disciplina **Engenharia de Dados e Big Data**.

## Objetivo

O objetivo deste projeto é aplicar conceitos relacionados a:

- Consumo de dados a partir de APIs
- Manipulação e tratamento de dados em formato JSON
- Persistência de dados em bancos de dados relacionais
- Modelagem de dados para armazenamento e consulta eficiente
- Integração de dados para uso em aplicações futuras

## Estrutura do Projeto

O repositório contém:

- **Notebook Jupyter (.ipynb)** com a implementação e explicações do processo
- **Arquivo JSON** utilizado como exemplo de resposta de API
- **Scripts ou trechos de código** para modelagem e persistência dos dados em banco relacional

## Tecnologias Utilizadas

- Python
- Jupyter Notebook
- JSON
- SQL
- Banco de dados relacional (ex: SQLite)

## Contexto Acadêmico

Este projeto foi desenvolvido como parte das atividades da disciplina **Engenharia de Dados e Big Data**, com o objetivo de demonstrar práticas de ingestão, modelagem e armazenamento de dados provenientes de APIs.
